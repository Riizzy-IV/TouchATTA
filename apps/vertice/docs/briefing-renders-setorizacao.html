Briefing — Renders de Setorização

Vértice Anália Franco · Para envio ao escritório 3D / Visualização

1. O que é a Setorização

A tela de Setorização mostra o edifício ao lado de uma lista de zonas (Residencial, Studios, Lazer, Lobby, etc.). Quando o usuário toca numa zona, a área correspondente do edifício acende — o restante escurece ou permanece neutro.

Para funcionar de forma simples, precisamos de uma imagem neutra + uma imagem por zona, todas do mesmo ângulo, para simplesmente trocar a imagem conforme a zona selecionada. Não é SVG, não é 3D em tempo real — apenas troca de PNG.

💡 Lógica: ao clicar em "Unidades Residenciais", a imagem setor_residencial.png substitui a imagem base. Ao clicar novamente, volta para setor_base.png. Zero programação especial — só a imagem certa no lugar certo.

4. Como fazer cada imagem de zona

Para cada zona, o render deve seguir estas regras:

✗ Errado — trocar textura inteira
FACHADA
COM COR
DIFERENTE

Não mudar a cor da fachada inteira — parece errado e pouco profissional.

✓ Certo — overlay luminoso sobre a zona

Zona recebe um brilho/glow colorido, restante do edifício levemente escurecido. A fachada original permanece visível por baixo.

Passos técnicos no software 3D / compositing:

  1. Renderizar a imagem base com iluminação padrão (sem pós-processamento extra).
  2. Para cada zona: duplicar o render base e aplicar um vignette/escurecimento geral (algo entre 30–50% de opacidade).
  3. Sobre a área da zona específica: remover o escurecimento e adicionar um glow colorido (bloom / emissão de luz) — a cor por zona está na tabela acima.
  4. Exportar como PNG com fundo escuro (não transparente).
  5. Todos os 8 arquivos devem ter exatamente o mesmo enquadramento e proporção.
💡 Alternativa mais simples (se não houver pós-processamento): renderizar com iluminação noturna, e para cada zona, acender as janelas/áreas daquele pavimento com luz interna colorida. O resultado é igualmente eficaz e mais realista.

5. Especificações técnicas do arquivo

Formato
PNG ou JPG
JPG se o fundo for sempre escuro (menor tamanho). PNG se quiser fundo transparente.
Resolução
1920 × 1080 px
Ou 2560 × 1440 px para telas de alta densidade.
Proporção
16 : 9
Obrigatório — todas as imagens com a mesma proporção.
Ângulo
Frontal + leve 3/4
Um único ângulo para todas as imagens — não mudar entre zonas.
Fundo
Escuro (#071820)
Combinar com o fundo da interface do app.
Nomenclatura
setor_[zona].png
Conforme tabela da seção 3.
